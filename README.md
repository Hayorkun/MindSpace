# MindSpace 🧠

> Clear your mind. Own your day.

**MindSpace** is a minimalist, friction-free task management application designed for cognitive offloading. Instead of overwhelming users with complex hierarchies and aggressive alerts, MindSpace acts as an external brain—allowing you to instantly dump your thoughts, reduce mental clutter, and focus on what matters right now.

## ✨ Features

*   **Instant Brain Dump:** A frictionless entry point designed to catch thoughts before they slip away.
*   **Focus Mode:** Filter out the noise and display only the immediate next task to maximize deep work.
*   **Calming UI:** A clean, spacious, low-contrast interface engineered to reduce digital anxiety.
*   **Real-time Persistence:** Seamless data synchronization ensuring your mental space is preserved across sessions.

## 🚀 Tech Stack

*   **Frontend:** React (Vite)
*   **Styling:** Tailwind CSS (for an elegant, utility-first fluid design)
*   **Icons:** Lucide React
*   **State & Database:** Express.js
